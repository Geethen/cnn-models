# Deep Learning with Keras

Implememnation of Deep Learning models in keras. 
***

## Getting Started

### Prerequisites

* Keras 2.2.2
* TensorFlow 1.10.0

```shell
pip install --upgrade keras
pip install --upgrade tensorflow
pip install --upgrade tensorflow-gpu
```
